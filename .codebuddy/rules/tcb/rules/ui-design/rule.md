# UI 设计规则 - 幻兽养成班级积分系统

> 版本：v1.0
> 适用项目：teacher-tool (班级积分管理系统)

---

## 一、设计风格

### 1.1 整体基调
- **游戏化视觉**：活泼、明亮、有趣、童趣
- **风格定位**：3D 萌系、盲盒玩具风、Q版可爱
- **目标用户**：小学生 + 老师

### 1.2 设计关键词
- 柔和、圆润、温暖
- 卡片式布局
- 马卡龙色系
- 轻量动效

## 七、组件规范

### 7.1 学生卡片
```
┌─────────────────────────┐
│  [幻兽图片]              │
│  ────────────           │
│  学生姓名    Lv.5       │
│  幻兽名称               │
│  ▓▓▓▓▓▓▓░░░ 35/50     │
│  🏅 徽章 x 3            │
└─────────────────────────┘
```

- 宽度：160-200px（响应式）
- 高度：自适应
- 圆角：16px
- 背景：白色
- 阴影：默认卡片阴影

### 7.2 按钮样式

**主按钮**
- 背景：渐变（珊瑚橙 → 樱花粉）
- 文字：白色
- 圆角：8px
- 高度：40px

**次按钮**
- 背景：白色
- 边框：1px 主色
- 文字：主色
- 圆角：8px

**幽灵按钮**
- 背景：透明
- 文字：次文字色
- 圆角：8px

### 7.3 进度条
- 高度：8px
- 圆角：4px
- 背景：#E5E5E5
- 填充：渐变（根据等级变化）
- 动画：填充时带缓动

### 7.4 徽章/标签
- 圆角：胶囊形 (9999px) 或 4px
- 内边距：4px 8px
- 字号：12px
- 颜色：根据等级/稀有度

## 八、动效规范

### 8.1 基础过渡
```css
transition: all 0.2s ease;
```

### 8.2 动效场景

| 场景 | 动效 |
|------|------|
| 卡片悬停 | 轻微上浮 + 阴影加深 |
| 按钮点击 | 缩放 0.95 |
| 页面切换 | 淡入淡出 |
| 升级 | 礼花/星星粒子特效 |
| 加分 | 食物飞入 + 数字弹出 |
| 进度条 | 平滑填充动画 |

### 8.3 Framer Motion 推荐配置
```javascript
// 弹性动画
const spring = {
  type: "spring",
  stiffness: 300,
  damping: 20
};

// 淡入
const fadeIn = {
  initial: { opacity: 0, y: 10 },
  animate: { opacity: 1, y: 0 },
  transition: { duration: 0.2 }
};
```

## 十三、设计原则

1. **一致性**：相同功能使用相同的视觉表现
2. **反馈性**：每个操作都有明确的视觉反馈
3. **可读性**：文字清晰可辨，信息层次分明
4. **趣味性**：保持游戏化氛围，让学生感到有趣
5. **简洁性**：避免视觉噪音，突出核心内容
