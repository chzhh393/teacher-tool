# 幻兽学院 — 订阅版本与定价方案

## 背景

产品现状：0.9元激活码已售400-500份，需要设计正式的订阅定价体系。目标是先跑量到2000-3000用户（基础版），再转化为付费升级用户。

**关键策略：基础版不做纯免费。** 所有渠道统一 ¥0.9 低价门槛：
1. **小红书渠道**：下单拿到手机号 + 发货时加微信 → 建立私域
2. **产品内渠道**：微信支付 ¥0.9 → 降低进入门槛，分享/搜索来的用户也能付费进入
3. **筛选真实意向用户**：愿意花 0.9 的人才是目标用户
4. 后续升级推送：小红书用户走私域，产品内用户走站内通知

## 版本命名

| 版本 | 英文标识 | 价格 | 定位 |
|------|---------|------|------|
| **基础版** | basic | ¥0.9 | 低价获客，建立私域，体验核心流程 |
| **会员版** | pro | ¥19.9起 | 重度使用、多班级、协作需求 |

> 只做两个版本，决策简单，转化路径清晰。
> 基础版 ¥0.9 不是为了收入，而是为了拿到联系方式、建立私域关系。

## 功能划分

### 基础版（¥0.9）

> **两条购买渠道：**
> - 小红书 → ¥0.9 下单 → 拿到激活码 → 注册激活（顺便拿到手机号、加微信）
> - 产品内 → 注册 → 微信支付 ¥0.9 → 自动激活（联系方式后续再收集）
> - 老师分享 → 打开链接 → 走产品内支付流程
>
> 不设使用期限，永久可用。

**班级管理**
- 1 个班级
- 最多 50 名学生
- 学生增删改、Excel 导入

**幻兽养成**
- 30 只基础幻兽（梦幻系15 + 热血系15）
- 完整的 1-10 级进化系统
- 徽章收集

**积分系统**
- 自定义积分规则（加分 / 扣分）
- 批量加减分
- 积分撤回
- 历史记录查看（最近 90 天）

**排行榜**
- 个人荣誉榜

> 设计原则：¥0.9 拿到联系方式，90天记录足够体验。让老师能体验幻兽养成核心流程，用起来之后自然想要更多（完整记录、课堂工具、多班级、导出、统计、小卖部），届时通过微信私域推送升级。

> **关于多账号绕过**：不主动封堵。多账号意味着数据分散、无法跨班统计、切换麻烦，体验远不如一个会员版。愿意折腾的是极少数，强堵反而伤口碑。

## 定价

### 小红书 SKU 一览

| SKU | 价格 | 版本 | 时长 | 定位 |
|-----|------|------|------|------|
| 基础版 | ¥0.9 | basic | 永久 | 获客入口，拿手机号+加微信 |
| 会员版·学期版 | ¥19.9 | pro | 180天 | 价格锚点，衬托年版 |
| 会员版·年版 ⭐ | ¥29.9 | pro | 365天 | 主推款，一杯奶茶心理价位 |
| 会员版·创始人买断 | ¥99 | pro | 永久 | 限量发售，含现有会员功能永久使用 |

### 产品内购买（微信支付）

产品内支持直接购买所有版本，价格与小红书一致，对接微信支付。

- 未注册用户 → 注册后弹出支付页 → 支付 ¥0.9 起 → 自动激活
- 已登录基础版用户 → 设置页/升级入口 → 支付升级为会员版
- 老师分享链接过来的新用户 → 同上，走产品内支付

**定价逻辑：**
- 小红书 ¥0.9 → 获取联系方式建立私域
- 产品内 ¥0.9 → 降低注册门槛，让分享/搜索来的用户也能付费进入
- 学期版存在的目的 → 让年版显得划算
- 年版是主推款 → 价格锚定在"一杯奶茶"心理价位
- 创始人买断 ≈ 3.3 年年费 → 满足买断心理，实际大概率不亏（老师换岗/换工具周期 < 3年）
- 创始人买断限量发售（限 2000 份或截止某日期），制造稀缺感
- 买断范围：✅ 现有会员功能永久使用 + 未来新增幻兽/主题皮肤 / ❌ 不含全新大模块（家长端、学校管理后台等单独定价）
- 当前所有价格标注"首发限时价"，后续可随用户量和功能增加逐步上调

## 老用户过渡策略

现有 0.9 元激活码用户（约 500 人）：
- 统一赠送**会员权益 1 年**（从激活日起算），清晰、可验证、好执行
- 1 年后自动降为基础版，但数据不删除、续费即恢复
- 到期前发专属优惠码：年版 ¥19.9（老用户价）/ 创始人买断 ¥69
- 这批用户是种子用户，转化成本要低，口碑传播价值远高于差价

## 教师裂变策略

### 核心机制：邀请返利

老师邀请老师，双方都有好处，形成自传播飞轮。

**邀请奖励方案：**

| 事件 | 邀请人获得 | 被邀请人获得 |
|------|-----------|-------------|
| 被邀请人购买基础版 | — | 基础版（正常流程） |
| 被邀请人升级会员版 | 延长30天会员时长 | 额外赠送15天会员时长 |
| 累计邀请3人付费 | 解锁"传奇驯兽师"称号 + 额外30天 | — |
| 累计邀请10人付费 | 当年免费使用会员版 | — |

**实现方式：**
- 每个会员版用户生成专属邀请码/链接
- 被邀请人注册时填写邀请码（或通过链接自动关联）
- 邀请记录在个人中心可查看，展示邀请进度条（"再邀请 1 位付费老师，送 30 天"）

**防刷规则：**
- 被邀请人会员付费满 **7 天不退款** 后，才给邀请人发放时长奖励
- 奖励分两段：立即解锁称号/徽章（即时反馈），时长延长 7 天后发放

### 为什么用"延长时长"而不是"返现"

1. **零成本** — 延长时长对你没有边际成本，返现要掏钱
2. **锁定用户** — 时长越长，用户迁移成本越高，续费概率越大
3. **简单合规** — 不涉及资金流转，不需要提现功能

### 社交传播触点

在产品内自然植入分享动机，不靠弹窗强推：

| 触点 | 场景 | 裂变方式 |
|------|------|---------|
| **班级分享页** | 老师分享班级进度给家长 | 页面底部"老师也在用幻兽学院"引导语 + 注册入口 |
| **光荣榜截图** | 老师发朋友圈/家长群晒排名 | 截图自带产品水印（基础版带水印，会员版可去除） |
| **学期末报告** | 学期总结数据导出 | 导出的PDF/图片带产品标识 |
| **课堂工具使用** | 投屏随机点名、静音挑战 | 工具界面角落显示"幻兽学院"品牌（低调不影响使用） |
| **个人中心** | 日常使用 | "邀请同事" 入口，展示邀请进度和奖励 |

### 裂变话术模板（供老师转发）

在产品内提供一键复制的推荐话术：

> "我最近在用一个班级管理工具，学生养幻兽特别有动力，推荐给你试试～ [链接] 用我的邀请码 XXXX 注册，付费时多送15天"

### 阶梯激励（加速裂变）

当裂变达到一定规模后，可以启动学校级裂变：

- **同校3位老师付费** → 该校所有老师享受 8 折
- **同校10位老师付费** → 学校管理员面板免费解锁（规划中的功能）

> 这个机制把单个老师的个人决策变成了学校级的集体决策，客单价乘以人数效果显著。

## 实施节奏（不含时间估算）

1. **阶段一：跑量 + 埋标识** — 维持现有 ¥0.9 模式全功能开放，但在会员功能入口加「会员」皇冠标识，潜移默化建立认知。积累到 2000-3000 用户，同时建立私域（加微信）
2. **阶段二：上线订阅** — 新用户 ¥0.9 购买基础版（功能受限），付费升级会员版；老用户赠送会员权益1年；小红书上架4个SKU
3. **阶段三：教师裂变** — 上线邀请返利机制，加速教师间传播
4. **阶段四：家长端** — 老师分享 → 家长注册查看 → 家长付费（独立定价）
5. **阶段五：周边商城** — 幻兽实物周边（后续再议）
