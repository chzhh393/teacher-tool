# 2026-02-10 批量模式粘贴姓名快速选人

## 本日目标

实现批量模式下通过粘贴学生姓名快速选中学生的功能，减少老师从列表中逐个查找学生的操作成本。

## 完成内容

- 新增 `parseSearchNames` 工具函数，支持按空格、逗号、顿号、分号、换行等分隔符拆分多个姓名
- 增强 `filteredStudents` memo，多名字输入时按"匹配任一"过滤学生列表
- 新增 `handleQuickSelect` 函数，匹配学生并追加到已选列表，显示选中/未匹配反馈
- 搜索框增强：批量模式下 Enter 键触发选中、旁边显示「选中」按钮、placeholder 提示多姓名输入
- PC 端搜索框加宽（`md:w-40` → `md:w-60`）

## 遇到的问题

无，实现较为顺利。方案设计阶段评估了多种交互方式（新增弹窗 vs 增强搜索框），最终选择增强搜索框方案，改动最小且操作步骤最少。

## 解决方案

复用现有搜索框和 notice 反馈机制，不新增组件。核心逻辑：
1. 搜索框输入多名字 → 实时过滤显示匹配学生
2. Enter/点击选中 → 将匹配学生追加到 `selectedIds`（Set 去重）→ 清空搜索 → notice 反馈
