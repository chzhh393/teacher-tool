# 幻兽提示词库 (Prompts Library)

> 本文件包含新增幻兽系列的图片生成提示词。
> 每条提示词仅包含**主体部分**，使用时需拼接统一前缀和后缀。

## 使用方法

```
前缀: 3D cute cartoon style, blind box toy style, C4D render, clean white background, high quality, 8k,
后缀: --no multiple views, no split screen, no text, no human
```

将主体提示词复制到 `scripts/generate-beasts/test.py` 的 `TEST_PROMPT` 中即可。
